## Array (String)
index-value pair
### Static & Dynamic
#### Static
- fixed inside
- Static arrays are allocated memory at compile time and the memory is allocated on the stack.
#### Dynamic
- the dynamic arrays are allocated memory at the runtime and the memory is allocated from heap
